# 开箱即用的h5模板

- [√] 🎉 rem配置，自动转换px为rem；
- [√] 🍀 集成service-worker；
- [√] 🏁 axios二次封装；
- [√] 💃🏻 pinia+pinia-plugin-persist实现状态管理，数据持久化；
- [√] 🛠 图片懒加载v-lazy指令；
- [√] 🎂 CryptoJS加密图片解密，返回base64/blob；
- [√] 🛠 构建优化；
- [√] 🐱‍🐉 多模块集成，多项目共用一份配置，可以互相依赖，也可以独立打包部署
